# Brew CheetSheet

## Brew install
~~~
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
~~~

## Basic Command

``$ brew install something``  
``$ brew update (something)``  
``$ brew upgrade something``  
``$ brew remove something``  
``$ brew list ``  
``$ brew upgrade (something)``
